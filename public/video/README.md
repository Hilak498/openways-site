# Logo video assets

Place the converted logo animation files here (see the root README for the
exact ffmpeg commands):

- `logo-animation.webm` — VP9 with alpha (Chrome / Firefox / Edge)
- `logo-animation.mov` — HEVC with alpha (Safari)
- `logo-poster.png` — poster frame (first or final frame)

Until these files exist, the hero automatically falls back to the static logo.
